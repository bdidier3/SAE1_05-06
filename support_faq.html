Se connecter à son Espace Client

  1. Visitez le site officiel de SFR.
  2. Cliquez sur "Se Connecter" en haut à droite de la page d'accueil.
  3. Remplissez les informations demandées, identifiant/mail ainsi que votre mot de passe.
  4. Acceptez les conditions d'utilisation et cliquez sur "Valider".
  5. Un email de confirmation sera envoyé à votre adresse.

Support et Dépannage SFR

  1. Rendez-vous sur le site de SFR.
  2. Cliquez sur l'onglet "Assistance" dans le menu principal.
  3. Choisissez le type de service pour lequel vous avez besoin d'aide (Internet, Mobile, etc.).
  4. Consultez les articles de la FAQ ou les guides de dépannage proposés.
  5. Si vous ne trouvez pas la solution, cliquez sur "Nous contacter" pour obtenir de l'aide par chat, email ou téléphone.

Question Générale


Pour souscrire à une offre SFR, vous pouvez vous rendre sur le site officiel de SFR, sélectionner l’offre qui vous convient et suivre les étapes de souscription en ligne. Vous pouvez aussi vous rendre dans une boutique SFR ou appeler le service client.

Pour résilier votre abonnement SFR, vous devez contacter le service client SFR par téléphone ou envoyer une lettre de résiliation à l’adresse indiquée dans vos conditions d’abonnement. Les frais de résiliation peuvent s’appliquer selon les conditions de votre contrat.

La Box SFR inclut un accès à Internet haut débit, la télévision avec plusieurs chaînes incluses, ainsi que la téléphonie fixe illimitée vers de nombreuses destinations. Des options supplémentaires comme SFR Cloud ou SFR Sécurité peuvent également être disponibles selon l'offre.

Après l'installation de votre box SFR, vous pouvez suivre les instructions fournies dans le guide d'installation ou sur le site de SFR. Vous aurez également accès à un espace de configuration en ligne pour personnaliser vos réglages Wi-Fi et services.

Vous pouvez contacter le service client SFR en appelant le 1023 depuis une ligne SFR ou via votre espace client en ligne. Le service client est également disponible sur les réseaux sociaux et par chat sur le site de SFR.
